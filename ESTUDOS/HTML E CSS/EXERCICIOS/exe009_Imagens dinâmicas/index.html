Imagens dinâmicas

Como usar imagens dinâmicas

Tente abrir esse asite em vários dispositivos diferentes ou aumente e diminua o tamando do seu navegador

Para a imagem ir diminuindo de acordo com a tela do dispositivo você usa a tag PICTURE, adiciona a imagem dentro dessa tag, pode usar a tag source-media-type, ela inicia com min-width é melhor mudar para max. dessa forma você aplica a imagem maior e depois as menores.

Ao aumentarmos a janela de visualização observamos a mudança do tamanho da imagem
